# BloomU
## Goals minggu 1
### Inisialisasi Proyek
* Membuat repository GitHub ✅
* Melakukan setup project sesuai dengan framework yang digunakan ✅
* Menentukan code convention, mekanisme branching, commit message, dan PR review ✅
### Minimal slicing halaman Register, Login, dan Home Screen
* Mengimplementasikan tampilan UI berdasarkan desain dari tim UI/UX ✅
* Menggunakan widget atau komponen yang reusable ✅
### Setup navigasi antar halaman
* Mengimplementasikan sistem navigasi yang sesuai dengan framework yang digunakan ✅
* Memastikan navigasi antar halaman berjalan dengan lancar, termasuk handling back navigation ✅
* Menentukan struktur routing yang mudah dipahami dan scalable ✅
### Setup State Management
* Memilih state management yang sesuai dengan framework yang digunakan ✅
* Mengelola data global seperti status autentikasi ✅
### Implementasi mekanisme autentikasi (Register dan Login)
* Implementasi autentikasi menggunakan Email/Password, Google Sign-In, atau metode lain yang disepakati ✅ 
* Validasi data pada input form (contoh: email harus valid, password minimal 8 karakter) ✅
* Menampilkan pesan error yang jelas saat autentikasi gagal ✅

## Goals minggu 2

### Menyelesaikan 75% slicing halaman yang sudah dibuat oleh UI/UX
* Menyelesaikan 75% dari seluruh halaman UI yang telah dirancang oleh tim UI/UX.
* Memastikan tampilan sesuai desain dan dapat beradaptasi dengan berbagai ukuran layar.

### Menyelesaikan minimal 1 fitur MVP selain Login dan Register
* Mengembangkan minimal 1 fitur utama aplikasi selain autentikasi. Contoh fitur: Komunitas, Chat, Notifikasi, atau fitur utama aplikasi.
* Memastikan validasi dan error handling sudah diterapkan dengan baik.
